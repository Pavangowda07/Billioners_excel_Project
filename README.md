# 📊 Billionaires Data Analysis Dashboard (Excel)

## 📌 Project Overview

This project is an interactive data analysis dashboard built entirely in **Microsoft Excel** using a global Billionaires dataset.

The objective of this project was to perform end-to-end data analysis including:

* Data Cleaning
* Descriptive Analysis
* Pivot Table Analysis
* Interactive Dashboard Creation

The dashboard provides insights into wealth distribution, industry trends, demographic segmentation, and country-wise billionaire concentration.

---

## 🧹 1. Data Cleaning

The raw dataset required preprocessing before analysis. The following cleaning steps were performed:

* Removed duplicate records
* Standardized the **Gender** column using Find & Replace
* Calculated Age using `YEARFRAC()` function
* Reorganized columns using Shift + Drag
* Cleaned GDP column (removed commas and converted to numeric format)
* Applied AutoFit formatting (`Alt + H + O + I`, `Alt + H + O + A`)

These steps ensured data consistency and improved accuracy for analysis.

---

## 📊 2. Data Analysis

### 🔹 Descriptive Statistics

Performed basic statistical analysis including:

* Mean
* Median
* Sum
* Count
* Average

This provided a high-level overview of wealth distribution.

---

### 🔹 Pivot Table Analysis

Created multiple Pivot Tables to answer business-focused questions:

* 🔝 Top 10 Richest Persons
* 👥 Billionaires by Age Range
* 🌍 Top 10 Countries by Number of Billionaires
* 🏭 Top Industries by Total Net Worth
* 💼 Percentage of Self-Made Billionaires

---

## 📈 3. Dashboard & Visualization

An interactive dashboard was developed using:

* Bar Charts
* Clustered Column Charts
* Pie Charts
* Slicers (connected using Report Connections)
* Dynamic Pivot Charts

The dashboard updates automatically based on slicer selections such as:

* Gender
* Self-Made Status
* Industry
